# Contributing

Thanks for your interest in contributing to this project.

Contributions are managed centrally for the Phenotype organization. Please
read the org-wide contribution guide before opening an issue or pull request:

- Org guide: [/Phenotype/CONTRIBUTING.md](https://github.com/KooshaPari/Phenotype/blob/main/CONTRIBUTING.md)

That document covers:

- Code of conduct
- Development setup
- Pull request process and review expectations
- Coding standards and quality gates
- Security reporting (cross-link to `SECURITY.md`)
